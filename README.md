# 💙 Heart Disease Prediction App

A Machine Learning web application that predicts the risk of heart disease based on user health inputs. Built using **Streamlit**, this app provides an interactive interface with a modern UI.

---

## 🚀 Features

* 🧠 Machine Learning prediction using KNN
* 🎨 Beautiful UI with background & glass effect
* ⚡ Real-time prediction
* 📊 Risk indication (High / Low)
* 🩺 User-friendly medical input form

---

## 🧠 Model Details

* Algorithm: K-Nearest Neighbors (KNN)
* Preprocessing: Standard Scaler
* Input Features:

  * Age
  * Sex
  * Chest Pain Type
  * Blood Pressure
  * Cholesterol
  * ECG Results
  * Max Heart Rate
  * Exercise-induced Angina
  * Oldpeak
  * ST Slope

---

## 🖥️ How to Run

### 1. Clone Repository

```bash
git clone https://github.com/your-username/heart-disease-prediction.git
cd heart-disease-prediction
```

### 2. Install Requirements

```bash
pip install -r requirements.txt
```

### 3. Run App

```bash
streamlit run app.py
```
